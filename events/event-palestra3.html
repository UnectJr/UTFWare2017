Título: Como as empresas estão inovando com Big Data

Local: Anfiteatro - UTFPR-CP

Palestrante:Rosangela Marquesone

Descrição: As oportunidades obtidas a partir da análise de grande volume de dados têm se mostrado cada vez mais atrativas no mundo dos negócios. Empresas estão desenvolvendo soluções guiadas por dados para aperfeiçoar seus produtos e serviços, para reduzir seus custos e enriquecer a experiência de seus clientes. Nessa palestra serão apresentados exemplos de como os dados estão sendo utilizados de forma inovadora nos últimos anos, demonstrando as estratégias e tecnologias adotadas para obter sucesso com Big Data.
