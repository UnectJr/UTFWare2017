Minicurso 4

Título: Implementação de um Sistema Wi-Fi em MatLab.

Palestrante: Dr. Lucas Dias Hiera Sampaio

Descrição: Utilizar a ferramenta MatLab para implementação da camada física de um sistema 802.11 incluindo blocos como modulador, demodulador, formatador de pulso, conversor A/D, codificador linear, equalizador, canal de transmissão, detector linear, entre outros.


Minicurso 5

Título: Técnicas de mineração de dados com R.

Palestrante: Rosangela Pereira Marquesone

Descrição: A extração de informações relevantes a partir de um conjunto de dados não ocorre em um momento “eureka”. Ao invés disso, torna-se necessário obter um conjunto de habilidades para capturar, transformar e analisar os dados corretamente. Nesse minicurso você irá conhecer um pouco mais sobre como a linguagem R pode ser utilizada no processo de mineração de dados. Para isso, iremos implementar duas soluções: um algoritmo para recomendação de músicas e um algoritmo para classificação de mensagens.
