Minicurso 3

Título: Programação criativa com Arduino e Processing

Palestrante: Henrique Stabile

Local: NIPAR (Ninho de Pardais).

Descrição: Workshop de prototipagem rápida de interfaces que integram o hardware do Arduino com a linguagem Processing de maneira criativa. O foco principal da oficina será na passagem de dados do Arduino para o Processing e do Processing para o Arduino, ou seja, o processamento de dados do físico para o digital e do digital para o físico, bem como da utilização destes dados para geração de imagens, vídeos e sons.
